# Git-10-hw1.1

# Название проекта

NeuroStartUp - динамически развивающийся стартап, специализирующийся на поиске с использованием новейших технологий искусственного интеллекта.

## Начало работы

Для того, чтобы получить копию этого проекта для запуска на локальном ПК необходимо в репозитории на [GitHub](https://github.com/Antis85/git-10-hw1.1) нажать на кнопку "Clone or download":

* далее нажать на кнопку "Download ZIP". Затем нужно указать путь для сохранения zip-архива на ПК и дождаться скачивания архива. После этого необходимо разархивировать скачанный zip-архив. 

или:

* скопировать URL репозитория и в Git выполнить команду `git clone https://github.com/Antis85/git-10-hw1.1.git`.

### Prerequisites

Для использования проекта нужно установить на ПК программу Git и браузер.

### Установка и запуск

Для запуска проекта требуется запустить файл index.html.
